هذا الدليل عملي ومباشر، بمثابة خارطة طريق سريعة لعالم الألعاب على لينكس. لا نظريات طويلة ولا تعقيد — فقط خطوات عملية لتبدأ اللعب فوراً. الأمثلة مبنية على **بوب أو إس (Pop!_OS)** لسهولته الاستثنائية.

### 🔍 مواقع لفحص توافق الألعاب (قبل التثبيت)

استخدم هذه المواقع لتعرف إن كانت لعبتك تعمل على لينكس بسلاسة أم تحتاج تعديلات بسيطة.

1. **ProtonDB** (الأهم لمستخدمي Steam)  

https://www.protondb.com

   #### ابحث عن اللعبة وراقب التقييم:  
   - **Platinum**: تعمل مباشرة  
   - **Gold**: تعمل مع إعداد بسيط  
   - **Silver**: تعمل مع بعض المشاكل  
   - **Bronze/Borked**: لا تعمل  

   *نصيحة: اقرأ تعليقات المستخدمين دائماً.*

2. **SteamDB** (معلومات تقنية)  

https://steamdb.info

   تحقق من أيقونة لينكس أو قسم Proton Compatibility.

3. **Are We Anti-Cheat Yet?** (لألعاب الأونلاين)  

https://areweanticheatyet.com

   يكشف ألعاب Fortnite، EA FC، Call of Duty التي تفشل بسبب مضاد الغش.

4. **Lutris Database** (للألعاب خارج Steam)  

https://lutris.net/games

   يعرض أفضل إعدادات التشغيل ونوع Wine المستخدم.

5. **PCGamingWiki** (موسوعة الحلول) 

https://www.pcgamingwiki.com/wiki/Linux

   قسم Linux يسرد المشاكل والحلول اليدوية.

### 🎮 إعداد المنصات (مثال: Pop!_OS)

**افتح الطرفية أولاً**: زر **Windows + T** أو ابحث "Terminal".

### 1. Steam
**التثبيت الصحيح:**  
1. افتح **Pop!_Shop** (أو Software Manager في توزيعات أخرى)  
2. ابحث **Steam**  
3. ⚠️ **اختر النسخة تحتها (System/Deb/Universe)** — تجنب Flatpak  
4. Install → شغّل → سجّل  

**تفعيل Proton (مترجم ألعاب ويندوز):**  

Steam → Settings → Compatibility → فعّل Steam Play لكل الألعاب → **Proton Experimental** → أعد تشغيل.

**لعبة لا تعمل؟** خصائص → Compatibility → فرض Proton 9 أو 8.

### 2. Epic Games (عبر Heroic)
1. من متجر البرامج → **Heroic Games Launcher** → Install  
2. سجّل بحساب Epic → Library → Install → Play  

**إذا فشلت**: إعدادات Heroic → فعّل DXVK/VKD3D → اختر **Proton-GE/Wine-GE** (تلقائي في الإصدارات الحديثة).

**⛔ لا تعمل حالياً (مضاد غش):** Valorant، Fortnite، Call of Duty الحديثة، EA FC.

### 3. Lutris (للألعاب خارج Steam و Epic)

**متى أحتاج Lutris؟**

- ألعاب Blizzard (Diablo / WoW)
    
- خدمة Ubisoft Connect
    
- ألعاب قديمة
    
- ألعاب Windows خارج المتاجر
    

**متى لا أحتاجه؟**

- لو اللعبة تعمل على Steam أو Heroic
    

**التثبيت:**  
Pop!_Shop → Lutris → Install

**التشغيل:**  
Lutris → Search → Install → Play

> ملاحظة: إن كنت مبتدئًا، استخدم Lutris فقط عند الحاجة.

### 🚀 تحسين الأداء

#### وضع الأداء  
أيقونة البطارية → **Performance**.

#### (يركّز موارد الجهاز على اللعبة) GameMode 
  

sudo apt install gamemode

#### تغيير في تشغيل اللعبة

في Steam: خصائص اللعبة → Launch Options →  
**انسخ بدقة**: `gamemoderun %command%`

**تنبيه كلمة السر**: مخفية تماماً (لا نجوم) — طبيعي، اضغط Enter.

#### منع إغلاق الشاشة  
Settings → Power → عطّل Screen Blank.

### ⚙️ إعدادات داخل الألعاب (موصى بها)
- **Fullscreen**: ON  
- **VSync**: OFF  
- **Motion Blur**: OFF  
- **FSR/DLSS**: ON (ترقية الدقة + إطارات أعلى)

### 🎮 ذراع التحكم  
أوصل Xbox/PS → Steam → Settings → Controller → **Enable Steam Input**.

---

### ⚠️ أخطاء شائعة تجنّبها
1. لا تثبّت Steam Flatpak + Deb معاً  
2. لا تحذف اللعبة عند أول فشل (جرب Proton مختلف)  
3. ثبت الألعاب على قرص سريع (EXT4/Btrfs، لا NTFS)  
4. انسخ الأوامر بدقة تامة (المسافات مهمة)
