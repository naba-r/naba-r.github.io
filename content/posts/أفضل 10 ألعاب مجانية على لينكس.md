 توجد اليوم العديد من الألعاب الممتازة على لينكس، بعضها يعمل بشكل أصلي (Native)، وبعضها يعمل عبر Proton من خلال Steam.



**كيف تُشغَّل الألعاب على لينكس؟**

هناك 4 طرق شائعة لتثبيت الألعاب:
| الطريقة              | ماذا تعني؟                          | مناسبة للمبتدئين؟ |
|----------------------|--------------------------------------|-------------------|
| **Steam**            | متجر ألعاب مشهور                    | ✅ الأفضل         |
| **مستودعات التوزيعة** | تثبيت مباشر من النظام               | ✅ سهلة           |
| **Flatpak**          | حزم تعمل على أغلب التوزيعات         | ✅ ممتازة         |
| **AppImage**         | ملف تشغيل مباشر بدون تثبيت          | ✅ سهلة جدًا      |
 

### ماذا يعني "كرت شاشة يدعم OpenGL أو Vulkan"؟

بعض الألعاب تذكر مصطلحات مثل:

* OpenGL
* Vulkan
* DirectX

هي **تقنيات (واجهات برمجية)** يستخدمها كرت الشاشة لتشغيل الرسوميات داخل اللعبة

المقصود ببساطة:

* إذا كان جهازك يشغّل فيديوهات HD وألعابًا خفيفة بشكل طبيعي فغالبًا ستعمل الألعاب البسيطة.
* الألعاب الثقيلة تحتاج كرت شاشة حديث نسبيًا.
* كلما كان كرت الشاشة أحدث، حصلت على أداء ورسوميات أفضل.


### ماذا تعني أنواع الألعاب؟

| المصطلح    | المعنى                                                  |
|    - |                   - |
| FPS        | لعبة إطلاق نار من منظور الشخص الأول                     |
| RPG        | لعبة تقمص أدوار وتطوير شخصية                            |
| RTS        | لعبة استراتيجية فورية تبني فيها قواعد وجيوش أثناء اللعب |
| MOBA       | لعبة جماعية تعتمد على فريقين وأبطال بقدرات مختلفة       |
| Survival   | لعبة بقاء                                               |
| Simulation | لعبة محاكاة واقعية                                      |


>
> بعض ألعاب الأونلاين لا تعمل على لينكس بسبب أنظمة الحماية (Anti-cheat)،
> لكن الألعاب الموجودة في هذا الدليل معروفة بأنها تعمل عادة بشكل جيد على لينكس.

# كيف تثبت الألعاب على لينكس؟

## تثبيت Steam

### Ubuntu / Debian

```
sudo apt install steam
```

### Arch Linux

```
sudo pacman -S steam
```

### Fedora

```
sudo dnf install steam
```

### opensuse

```
sudo zypper in steam
```

 

## تثبيت Steam عبر Flatpak

```
flatpak install flathub com.valvesoftware.Steam
```

 

# ما هو Proton؟

Proton أداة من Valve تسمح بتشغيل ألعاب ويندوز على لينكس بسهولة.

لتفعيله:

1. افتح Steam
2. ادخل إلى Settings
3. ثم Compatibility
4. فعّل:

   * Enable Steam Play
   * Enable Steam Play for all titles
